# Сводка Изменений: Система Блокировки Уроков

## 📋 Краткое Описание

Реализована полная система блокировки/разблокировки уроков с условием минимальной оценки 4.5 за предыдущий урок. Только Урок 1 доступен в начале, остальные 31 урок заблокирован.

## 🔄 Логика Работы

### Инициализация
1. Пользователь открывает `lesson_menu.tsx` с ID урока
2. Проверяется `isLessonUnlocked(lessonId)`
3. Если заблокирован: показать 🔐, disable кнопку, prepare модальное окно

### При Клике на Заблокированный Урок
1. Срабатывает `handleLockedLessonPress()`
2. Показывается модальное окно (modal)
3. Текст сообщения из `getLockMessageText(lessonId, lang)`
4. Кнопка "Понимаю" закрывает модал

### При Завершении Урока
1. Все 50 ячеек ответлены правильно
2. Вычисляется `finalScore = correct / 50 * 5`
3. Вызывается `tryUnlockNextLesson(lessonId, finalScore)`
4. Если `finalScore >= 4.5`: разблокируется урок N+1
5. Сохраняется в AsyncStorage
6. Переход на экран `lesson_complete`

### Следующий Визит
1. Пользователь открывает `lesson_menu.tsx` для урока N+1
2. `isLessonUnlocked(N+1)` возвращает true
3. Урок доступен без замка ✅
