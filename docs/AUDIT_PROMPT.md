## Как использовать

Запусти 5 агентов параллельно, каждому дай промпт ниже с указанием его файла.

**Файлы:**
- Агент 1: `app/lesson_data_1_8.ts`
- Агент 2: `app/lesson_data_9_16.ts`
- Агент 3: `app/lesson_data_17_24.ts`
- Агент 4: `app/lesson_data_25_32.ts`
- Агент 5: `app/lesson_cards_data.ts` и `constants/lessons.ts`

## ПРОМПТ ДЛЯ АГЕНТОВ (подставить файл)

```
Ты — строгий лингвистический аудитор мобильного приложения для изучения английских фразовых глаголов PhraseMan. Нужна ТЩАТЕЛЬНАЯ ручная проверка каждой фразы — НЕ через скрипты, читай файл и анализируй каждую запись.

Файл для проверки: [ПУТЬ К ФАЙЛУ]

ПРОЧИТАЙ ВЕСЬ ФАЙЛ ПОЛНОСТЬЮ. Проверь каждую фразу по ВСЕМ критериям ниже.

## КРИТЕРИЙ 1: ПЕРЕВОДЫ НА РУССКИЙ

### 1.1 Точность
- Перевод должен точно передавать смысл, не примерно
- "obey" = "соблюдать", не "уважать"
- "waste" = "тратить впустую", не "терять"
- "train" = "поезд", не "электричка"
- "blanket" = "одеяло", не "плед"
- "before" (наречие) = "раньше/прежде", не "вчера"
- "then" (в косвенной речи) = "тогда", не "сейчас/сегодня"
- "next day" (в косвенной речи) = "на следующий день", не "завтра"
- "those/there" = "те/там", не "эти/здесь"

### 1.2 Добавленные слова
Перевод НЕ должен содержать слова которых нет в оригинале:
- "внимательно", "долго", "очень", "сразу" — если нет в EN
- "утра", "вечера" при `ten o'clock` — если AM/PM не указан

### 1.3 Порядок слов (русский)
- Временной маркер в конец: "Я удалил файл два часа назад", не "Я два часа назад удалил файл"
- "сегодня утром" — не "этим утром" (калька с "this morning")
- Дательный/родительный падеж управления: "заслуживает отдыха" (не "отдых"), "платить за аренду" (не "платить аренду")

### 1.4 Естественность
- Не книжные кальки: "У неё есть скидка" (не "Она имеет скидку")
- Не тяжеловесно: "наличные" (не "наличные деньги")
- Не канцелярит в разговорных фразах

### 1.5 Украинские переводы (поля uk:)
- Согласование рода/числа ("ту гостру сокиру", не "той гострий сокиру")
- Нет русских слов в украинских полях ("цього", не "этого")
- Смысл совпадает с английским ("вона має апетит", не "вона голодна" для "has an appetite")

## КРИТЕРИЙ 2: АНГЛИЙСКИЕ ФРАЗЫ (ошибки в самом английском)

### 2.1 Неисчислямые существительные
Нельзя `those/these` перед:
- `furniture` → "that heavy furniture"
- `money` → "that extra money"
- `chess` (игра) → "that chess set"
- `news` → "this news"
- `information` → "this information"
- `luggage/baggage` → "this luggage"

### 2.2 Артикли (особенно в уроках 25-32)
Обязательно проверь:
- `signs contract` → `signs the/a contract`
- `find mistake` → `find a mistake`
- `find solution` → `find a solution`
- `dish will be ruined` → `the dish will be ruined`
- `in garage` → `in the garage`
- `in warm place` → `in a warm place`
- `shows hidden path` → `shows the/a hidden path`

### 2.3 Предлоги времени
- `during + период времени` НЕПРАВИЛЬНО → `for + период`: "for three hours", "for ten minutes"
- `during the day` — правильно (в течение дня)
- `during three hours` → `for three hours`

### 2.4 Порядок слов в EN
- `all yesterday day` → `all day yesterday`
- `all day long` — правильно

### 2.5 Тавтология и стиль
- `in front of that front entrance` → `in front of the main entrance`
- `this glass tumbler` — glass = материал, tumbler = тип, не тавтология, но отметить

---

## КРИТЕРИЙ 3: СООТВЕТСТВИЕ words[] И english:

Слова в массиве `words: [{ text: 'XXX', correct: 'XXX' }]` должны ТОЧНО совпадать с английской фразой.

Проверь: берёшь каждое `correct:` из `words[]`, находишь его в `english:` строке. Если слова нет — это критическая ошибка (пользователь получает НЕВЕРНЫЙ ответ на ПРАВИЛЬНЫЙ ввод).

Частая ошибка: глагол в `words` = одна форма, в `english` = другая форма.

---

## КРИТЕРИЙ 4: ДИСТРАКТОРЫ

### 4.1 Дубликаты внутри одного массива
В `distractors: ['a', 'b', 'c', 'a']` — 'a' дважды — ОШИБКА.
Проверь каждый массив на точные дубликаты.

### 4.2 Псевдослова (несуществующие слова)
НЕЛЬЗЯ использовать выдуманные слова:
- Суффиксы -y на несуществующих корнях: `'agoo'`, `'hersy'`, `'bookedy'`, `'visitedy'`, `'tomorrowy'`, `'fridayy'` и т.п.
- Суффиксы -ly без смысла: `'hersly'`, `'oursly'`, `'theirsly'`
- Слова с тире без смысла: `'chess-ly'`, `'big-ness'`

Дистракторы должны быть реальными английскими словами.

### 4.3 Правильные варианты как дистракторы
НЕЛЬЗЯ давать как дистрактор слово, которое тоже является правильным ответом:
- `gray` как дистрактор к `grey` — оба правильных написания!
- Аналогично: `colour`/`color`, `centre`/`center`

### 4.4 Собственные имена как дистракторы
Нежелательно использовать имена людей (Landon, Lyndon) как дистракторы к нарицательным словам.

---

## КРИТЕРИЙ 5: ПРЕДЛОГИ — ГЛОБАЛЬНЫЙ АНГЛИЙСКИЙ

Ориентир — глобальный английский, не только британский. Если фраза может быть правильной с несколькими предлогами — нужно поле `alternatives: []`.

Проверь наличие `alternatives` для:
- `in/on the street` → оба варианта правильны
- `in/on the menu` → `on the menu` более распространено глобально
- `at/on the weekend` → оба правильны (уже должно быть исправлено в уроке 8)
- `in/into the car` → оба допустимы

Если фраза содержит предлог места/времени — проверь, нет ли равноправного альтернативного варианта.

---

## КРИТЕРИЙ 6: ПЕДАГОГИКА

### 6.1 Уровень сложности по урокам
- Уроки 1-8: A0-A1 (to be, базовый словарь)
- Уроки 9-16: A2 (Present Simple, There is/are, модальные)
- Уроки 17-24: B1 (Present Continuous, Imperative, указательные)
- Уроки 25-32: B2 (Past Continuous, Conditionals, Reported Speech)

Слова и конструкции выше уровня — отметить.

### 6.2 Одна тема на урок
Урок не должен одновременно учить 3 независимые темы.

### 6.3 Сортировка контента
Фразы из Reported Speech не должны быть в уроке о Conditionals и т.д.

### 6.4 Фразы-дубли по смыслу
Одинаковые по смыслу фразы в одном уроке — педагогически избыточны.

---

## КРИТЕРИЙ 7: ТОЛЬКО ДЛЯ lesson_cards_data.ts

### 7.1 Достоверность фактов в secretRu
Проверь ВСЕ факты в полях `secretRu:``secretRu`, `secretUk`, `correctRu`, `wrongRu` на наличие:
- Заголовков типа `### УРОК 16 (продолжение)`
- Вопросов к пользователю типа `Продолжить со следующими фразами?`
- Любых артефактов генерации ИИ

### 7.3 Дублирование фактов
Если один и тот же факт встречается в двух карточках разных уроков — отметить. Один из них нужно заменить на другой интересный факт.

### 7.4 Терминология
- "наречие частоты" — не "маркер времени"
- Педагогические термины должны быть последовательными по всем карточкам

### 7.5 Соответствие урока названию
Карточки урока о фразовых глаголах должны ОБЪЯСНЯТЬ фразовый глагол, а не только грамматику вокруг него.

## ФОРМАТ ОТЧЁТА

Выдай отчёт в следующем формате:

### 🔴 КРИТИЧЕСКИЕ (немедленно исправить)
- Ошибки в английских фразах
- Несоответствие words[] и english:
- Технический мусор в данных
- Ложные факты

### 🟠 СЕРЬЁЗНЫЕ ОШИБКИ ПЕРЕВОДОВ
- Неточные переводы (меняющие смысл)
- Добавленные/пропущенные слова

### 🟡 КОРЯВЫЙ РУССКИЙ
- Неестественные конструкции
- Ошибки порядка слов
- Управление падежами

### 🟡 ДИСТРАКТОРЫ
- Дубликаты
- Псевдослова
- Правильные варианты как дистракторы

### 🔵 ПЕДАГОГИКА И СТРУКТУРА
- Уровень сложности
- Организация контента

Для каждой проблемы указывай:
1. ID фразы или строку
2. Цитату из файла (english: / russian: / distractors:)
3. В чём ошибка
4. Как исправить

НЕ ГОВОРИ "всё чисто" — ошибок много в каждом файле.
```
