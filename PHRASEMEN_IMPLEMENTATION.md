## Структура хранения

AsyncStorage ключ: `'phrasemen_state'`

```typescript
{
  balance: number;                // Текущий баланс
  totalEarned: number;            // Всего заработано
  totalSpent: number;             // Всего потрачено
  lastDailyBonus?: number;        // Timestamp последнего дневного бонуса
  transactions: Transaction[];    // История всех транзакций
}
```

Каждая транзакция содержит:
- `id` — уникальный ID
- `type` — тип (daily_task, streak_bonus, energy_purchase и т.д.)
- `amount` — сумма
- `reason` — описание (для UI)
- `timestamp` — когда произошла
- `isSpending` — это трата или заработок

## Примеры кода

### Использование в компоненте дневной задачи

```typescript
import { rewardPhrasemenForTask } from '@/app/phrasemen_integration';

export function DailyTaskItem({ task, onClaimed }: Props) {
  async function handleClaim() {
    // Отметить задачу как выполненную
    await claimTask(task.id);

    // Выдать фразмены
    await rewardPhrasemenForTask(task.id);

    // Обновить UI
    onClaimed?.(task.id);
  }

  return (
    <TouchableOpacity onPress={handleClaim}>
      <Text>{task.titleRU}</Text>
      <Text>XP: {task.xp} | ⭐: {task.phrasemenReward}</Text>
    </TouchableOpacity>
  );
}
```

### Использование в экране магазина

```typescript
import { SHOP_ITEMS } from '@/app/types/shop';
import { buyEnergy } from '@/app/phrasemen_integration';

export function ShopScreen() {
  async function handleBuyItem(item: ShopItem) {
    let success = false;

    if (item.category === 'energy') {
      const energyAmount = item.effect?.value || 5;
      success = await buyEnergy(energyAmount);
    }
    // ... другие категории

    if (success) {
      toast.success(`Куплено: ${item.titleRU}`);
      refresh(); // обновить баланс
    } else {
      toast.error('Недостаточно фразменов');
    }
  }

  return (
    <FlatList
      data={SHOP_ITEMS}
      renderItem={({ item }) => (
        <ShopCard
          item={item}
          onPress={() => handleBuyItem(item)}
        />
      )}
    />
  );
}
```
