# Сокращения: согласование data ↔ `lesson1.tsx`

Логика в `app/lesson1.tsx` (расширение) + `CONTRACTION_MAP` / `EXPANSION_TO_CONTRACTION` в `lesson1_smart_options.ts`.

**Data-checklist по фразе:**

- Если в одном **слоте** стоит `She's`, следующий слот **не** `is` / `am` / `are` (см. также скрипт `contraction` в `lesson:qa`).
- Если в **двух** слотах `She` + `is`, в опциях должны быть **альтернативы** с `She's` как в коде (не дублировать логику тут; проверяй *на устройстве* или по UX).
- **Не** смешивай в одной фразе взаимоисключающие разборы без педагогического плана (один путь: развернуто, другой: сокращение).

**Задача агента:** для каждой **спорной** фразы — один абзац: *ожидаемая последовательность нажатий* и что **не** должно появляться после выбора.

**Инструменты:** `lookupContraction`, `getPhraseWords` (импорты в скриптах при необходимости).
