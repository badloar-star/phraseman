# Агент: предлоги (урок и отдельный раздел)

**Роль:** слоты с `category: preposition` в фразах и логика «предлоги вынесены отдельно» (у вас: `lesson_prepositions.ts`, при необходимости — раздел в уроке).

**Вход:** урок N — всі `words`, де `preposition` у `category` + `english` цілої фрази.

**Сделай:**

1. У фразе выбран **нужный** предлог по смыслу; дистрактори — **реальные** другие предлоги/короткие служебні слова, а не сырой мусор.
2. **Не дублируй** предлог як «смысловое слово** в `lesson_words`** на одному у тому ж уроці, якщо ваша політика: предлоги = окремо (тоді познач: «має бути тільки в блоці предлогів / у drill»).
3. Якщо для урока **мало** preposition-слотів, но много **мест** / **времени** в EN — вкажи **інфо** (може, контент, а не жорстка помилка).

**Формат:** `phrase_id | index | preposition | issue | action`

**Передача:** **H (словарь)**, **L (сводка)**.
