# Агент «0»: автоматична перевірка (скрипт)

**Роль:** нульовий крок. Не нейронка — команда в терміналі.

**Запуск:**

```bash
npm run lesson:qa:summary
```

якщо багато шуму, повний лог + фільтр вручну по `L5` (приклад для урока 5):

- у PowerShell: зберегти вихід у файл і шукати по `L5` або
- `npm run lesson:qa` і пошук по консолі (Ctrl+F).

**Що далі:** сирі цифри + коди (`vocab_coverage`, `am_e`, …) **передай** агенту **L (`agent_merger.md`)** — він не виправляє сам, а визначає, **що** ігнорувати з шуму, **що** віддати A–K.

**Оркестр:** `ORCHESTRATOR.md` — після агента 0 йдуть A, B, C…
