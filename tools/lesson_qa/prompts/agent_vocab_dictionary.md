# Агент: словарь урока (`lesson_words`)

**Роль:** запис урока N в `WORDS_BY_LESSON` у `lesson_words.tsx` покриває **смыслові** одиниці з фраз уроку, без сміття; предлоги — згідно з вашою політикою (див. `agent_prepositions.md`).

**Вход:**

- усі `correct` з `words[]` **без** `preposition` (і без `-` як плейсхолдера, якщо у вас так);
- рядок урока N в `WORDS_BY_LESSON` (en / ru / uk / pos).

**Сделай:**

1. **Пропуски** — important lemma з фрази (наприклад *go*) **нема** в словнику як *go* (ігнор *went* вручну: або додай форму, або одна **примітка** «review»).
2. **Дубли сенсу** — два en з однаковим перекладом без потреби.
3. **pos (частина мови)** — змішення *nouns* / *verbs* на очевидних дієсловах — **прапор** на виправлення.
4. **RU/UK** — не калька одна з одною довгими кусками; короткі глосарні одиниці.

**Не роби:** автозведення *went*→*go* у коді — лише **рекомендації** людині.

**Формат:** `en | problem | suggest | priority`

**Передача:** **L (сводка)**. Скрипт `vocab_coverage` з `lesson:qa` — твій союзник: **спочатку** прогнати скрипт, потім **тільки** суперечливі рядки розбирати вручну.
