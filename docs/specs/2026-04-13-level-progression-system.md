## ФИЧА E: LEVEL-UP ПОДАРОК (VARIABLE REWARD)

### Психология

Variable reward = переменное подкрепление (Skinner, 1938). Именно на этом работают
игровые автоматы, TikTok, Instagram. Непредсказуемость создаёт БОЛЬШИЙ дофаминовый
выброс, чем фиксированная награда. Это самый сильный механизм удержания.

### Пул подарков

**COMMON (частые, 70% / 60% / 55% / 50% по уровням):**
| ID | Подарок | Описание |
|----|---------|----------|
| `energy_full` | ⚡ Полная энергия | Восстанавливает все слоты прямо сейчас |
| `xp_50` | 🎯 +50 XP | Мгновенные 50 XP |
| `xp_100` | 🎯 +100 XP | Мгновенные 100 XP |
| `hint_bonus` | 💬 +2 подсказки | Дополнительные подсказки для следующего урока |

**RARE (редкие, 25% / 30% / 30% / 30% по уровням):**
| ID | Подарок | Описание |
|----|---------|----------|
| `xp_2x_24h` | 🔥 2× XP на 24 часа | Удвоение XP за все активности |
| `energy_2x` | ⚡⚡ +2 энергии сейчас | Два дополнительных слота на сегодня |
| `streak_shield_1` | 🛡️ Стрик-щит × 1 | Один пропуск не сломает стрик |

**EPIC (очень редкие, 5% / 10% / 15% / 20% по уровням):**
| ID | Подарок | Описание |
|----|---------|----------|
| `xp_3x_48h` | 🚀 3× XP на 48 часов | Тройное XP на 2 дня |
| `streak_shield_3` | 🛡️🛡️🛡️ Стрик-щит × 3 | Три пропуска не сломают стрик |
| `wager_discount` | 💎 Ставка пари −50% | Следующее пари стоит вдвое дешевле |
