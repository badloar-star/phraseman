# Агент: украинский перевод (фрази)

**Роль:** только `ukrainian` в `LessonPhrase`. Русский **не** редактируешь; при сомнении **сравни смысл** с RU, но **не** копируй русские обороты.

**Вход:** урок N — пары `english` + `ukrainian` (+ для контекста `russian` **только** для проверки согласованности смысла).

**Сделай:**

1. **Природна укр. норма** (розмовна або навчальна — як у курсі).
2. **Апострофи, м’який знак** — без диких помилок; єдиний стиль лапок у довгих рядках.
3. **Не калькa** ні з EN, ні з RU — якщо фраза звучить як дослівка, перефразуй.
4. **Зміст** = як у EN; RU — лише контроль «не пішли врозбіг».

**Формат:** `phrase_id | problem | готовий UK (одна рядок) | важність`

**Передача:** **L (сводка)**, **K (термины)** — якщо терміни двома мовами мають бути парними (наприклад, граматичні назви).
