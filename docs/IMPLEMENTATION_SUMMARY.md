## Ключевые особенности

### 🎯 Персональные планы
Каждый пользователь получает уникальный план на основе:
- Цели обучения
- Доступного времени в день
- Текущего уровня английского
- Интенсивности занятий

**Пример расчёта:**
```
Цель: Туризм
Интенсивность: 15 минут в день
От A1 до B1

Прогноз:
- ~90 дней достижения
- 32 урока в твоём темпе
- ~2 часа в неделю обучения
- Целевой уровень к June 2026
```

### 📊 Интеллектуальные расчёты
```
Формула расчёта дней = baseDaysPerLevel × levelsDiff × intensityMultiplier

Коэффициенты интенсивности:
- 5 мин/день: ×3.0 (в 3 раза дольше)
- 15 мин/день: ×1.0 (базовая)
- 30 мин/день: ×0.6 (в 1.67× быстрее)
- 60+ мин/день: ×0.4 (в 2.5× быстрее)
```

### 🌍 Мультиязычность
- Полная поддержка русского и украинского
- Правильное использование грамматики и лексики
- Локализированное форматирование дат

### 🔄 Интеграция с существующей системой
- Сохранение в AsyncStorage (как остальные данные)
- Последовательность: beta → lang → name → **goal → minutes → level → plan → time** → test_offer → premium
- Совместимо с диагностическим тестом и Premium предложением

## Переходная логика (Flow)

```
1. beta (бета-информация)
   ↓
2. lang (выбор языка)
   ↓
3. name (имя пользователя)
   ↓

   ↓
5. minutes (⏱️ интенсивность)      ← НОВОЕ
   ↓
6. level (📊 текущий уровень)     ← НОВОЕ
   ↓
7. plan (📈 персональный план)    ← НОВОЕ
   ↓

   ↓

```
