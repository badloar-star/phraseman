### 1.3 🔵 Энергия: Базовая система
- **Что:** Система энергии вместо "жизней"
- **Статус:** АРХИТЕКТУРА
- **Приоритет:** 🔴 ВЫСОКИЙ — основа остального

**Параметры энергии:**
```typescript
interface EnergyState {
  current: number;        // 0-5
  max: number;           // 5
  recoveryPerHour: 1;    // восстанавливается 1 unit/2 часа
  lastRecoveryTime: number;
  totalEnergySpentToday: number;
}
```

**Где тратится энергия:**
- Урок: -1 энергия
- Ошибка в уроке: -0 энергия (но стрик сбрасывается)
- Квиз: -0 энергия (просто для удовольствия)
- Словарь: -0 энергия
- Неправильный глагол: -0 энергия

**Как восстанавливается:**
- Автоматически: +1 каждые 2 часа
- Премиум подписка: +1 каждый час
- Просмотр рекламы: +1 (раз в день)
- Покупка в магазине: +5 за монеты

**UI показ энергии:**
- На главном экране большой прогресс-бар с осталось сколько уроков
- Уведомление когда восстановилась энергия

**Задачи:**
- [ ] Создать `app/energy_system.ts`
- [ ] Интегрировать в `lesson1.tsx`
- [ ] UI компонент на `home.tsx`
- [ ] AsyncStorage для сохранения

### 1.4 🔵 Валюта: "Фразмены" (⭐ Оригинальное имя)
- **Что:** Вместо скучных "монет" или "алмазов"
- **Статус:** КОНЦЕПЦИЯ
- **Приоритет:** 🟡 СРЕДНИЙ

**Почему "Фразмены"?**
- Привязано к названию приложения Phraseman
- Меморабельно, уникально, не как у конкурентов
- Можно игровое происхождение: "древние мудрецы оставили фразмены за умные ответы"

**Как получать:**
- Дневные задачи: 5-20 фразменов за задачу
- Бонус за серии: +10 за 7-дневный стрик
- Сундук дневный: 15-30 фразменов
- Просмотр рекламы: +5
- Реферал: +50 за приглашённого друга

**Как тратить:**
- Восстановление энергии: 10 фразменов = +5 энергии
- Бустер x2 XP: 20 фразменов = 30 минут x2
- Рамка для профиля: 50-100 фразменов
- Премиум (если не через подписку): 99 фразменов

**Задачи:**
- [ ] Создать `app/phrasemen_system.ts`
- [ ] Сохранять баланс в AsyncStorage
- [ ] Анимация при получении фразменов

## Phase 2: Variable Reward (Неделя 3-4)

### 2.1 🟠 Случайный бонус XP после урока
- **Что:** Post-lesson slot machine
- **Психология:** Variable Reward (Скиннер) — непредсказуемость создаёт дофамин
- **Статус:** ДИЗАЙН

**Механика:**
```
После завершения урока с оценкой ≥ 4.5:
- 70% шанс: ×1 (обычный XP)
- 25% шанс: ×1.5 (бонус)
- 4% шанс: ×2 (редко)
- 1% шанс: ×3 + 10 фразменов (крайне редко) 🎉
```

**Анимация:**
- Экран с вращающимся колесом
- "Ты выиграл ×2 XP!" с конфетти
- Случайный звук успеха

### 2.3 🟠 Push-уведомления
- **Что:** Фраза дня + напоминание о стрике
- **Психология:** Loss Aversion — потеря стрика болит
- **Статус:** ПЛАНИРОВАНИЕ

**Типы пушей:**
1. **Фраза дня** (8:00 утра по preferred time)
   - "Сегодняшняя фраза: 'I went to the office yesterday'"
   - Tap → открывает урок с этой фразой

2. **Напоминание о стрике** (20:00)
   - "Твой стрик: 7 дней 🔥 Занимайся сегодня, чтобы не потерять!"
   - Tap → открывает домой

3. **Стрик сгорит через 3 часа** (21:00)
   - "⚠️ Осталось 3 часа до конца дня. Твой стрик сгорит!"
   - Loss aversion максимум

4. **Энергия восстановилась** (по мере восстановления)
   - "⚡ У тебя 5 энергии! Поехали учить!"

5. **Уровень на 3 XP от повышения** (когда близко)
   - "🎯 Всего 3 XP до уровня 15! Почти!"

- [ ] Сохранять `notificationTime` в профиле

**Отслеживание:**

## Психологические принципы в коде

### Zeigarnik Effect (незавершённое давит)
```typescript
// ВСЕГДА показывать прогресс, никогда не "пусто"
// ❌ НЕПРАВИЛЬНО: Progress 0/10 → 0 нового прогресса
// ✅ ПРАВИЛЬНО: Осталось 7/10 → кажется, что уже сделано много

- Daily tasks: "Осталось 2/3 заданий"
- Уровень: "3 XP от повышения (вместо нужно 97)"
- Стрик: "День 5 из 7" (вместо просто "5 дней")
- Урок: Прогресс-бар на 85% останавливается → "почти!"
```

### Loss Aversion (потеря > приобретение)
```typescript
// Страх потери сильнее радости приобретения в 2x раза

- Уведомление: "Твой стрик сгорит через 3 часа" (не "занимайся")
- UI: Красная статистика когда в опасности (3-е место в клубе)
- Energy: медленное восстановление → не хочешь тратить зря
- Пари на стрик: вложил → обязан продолжать
```

### Self-Determination (выбор + компетентность)
```typescript
// Три базовые потребности человека

АВТОНОМИЯ:
- Выбор пути: грамматика / словарь / разговорник (в онбординге)
- Выбор времени занятий
- Выбор интенсивности (5/15/30/60 мин в день)

КОМПЕТЕНТНОСТЬ:
- Видимый рост: "Сегодня ты знаешь на 12 слов больше"
- Система уровней (1-50) с видимой вертикалью
- Частые малые победы (ежедневные задачи)
- Прогресс-бары везде

ПРИНАДЛЕЖНОСТЬ:
- Клубы с атмосферой (у тебя 12 клубов, это 👍)
- Рейтинги (конкуренция с друзьями)
- Социальные механики (реферали, групповые бустеры)
```
