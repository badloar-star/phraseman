# Per-Word Hint System Demo

## Как работает новая система подсказок

### Пример 1: Одна ошибка

**Правильный ответ:** `I am a teacher`
**Ответ пользователя:** `I is a teacher`

**Что происходит:**
1. Система нормализует оба текста: `i am a teacher` vs `i is a teacher`
2. Сравнивает пословно:
   - Слово 0: `i` == `i` ✓
   - Слово 1: `is` != `am` ✗ → ищет WordTrap с wordIndex:1 → находит!
   - Слово 2: `a` == `a` ✓
   - Слово 3: `teacher` == `teacher` ✓

**Результат на экране:**
```
❌ Неправильно

Правильно: I am a teacher

[Синяя карточка с подсказкой]
С "I" глагол "to be" только "am": I am.

[Кнопка播放 для прослушивания]
```

### Пример 2: ДВЕ ошибки (главная фишка!)

**Правильный ответ:** `There is a book on the table`
**Ответ пользователя:** `There are the book in table`

**Что происходит:**
1. Нормализует: `there are the book in table` vs `there is a book on the table`
2. Сравнивает пословно:
   - Слово 0: `there` == `there` ✓
   - Слово 1: `are` != `is` ✗ → ищет WordTrap(wordIndex:1) → находит "There is для единственного числа"
   - Слово 2: `the` != `a` ✗ → ищет WordTrap(wordIndex:2) → находит "Артикль a перед согласной"
   - Слово 3: `book` == `book` ✓
   - Слово 4: `in` != `on` ✗ → ищет WordTrap(wordIndex:4) → не находит
   - Слово 5 (не существует в правильном) → пропускает

**Результат на экране (можно скролить!):**
```
❌ Неправильно

Правильно: There is a book on the table

[Синяя карточка #1]
"There is" для единственного числа (a book).

[Синяя карточка #2]
Артикль "a" перед согласной: a book.

[Кнопка播放]
```

## Технические детали реализации

5. **error_traps/error_traps_9_16.ts** - переделаны на per-word формат

### Структура WordTrap:

```typescript
interface WordTrap {
  wordIndex: number;   // позиция слова в правильном ответе (0-based)
  hint: string;        // подробное объяснение
  lite?: string;       // краткая версия для quiz-режима
}
```

### Fallback система:

- Если wordTraps не заданы → использует старую trigger-систему
- Если и trigger не работают → выводит generalRule
- Полностью обратно совместима со старыми error_traps файлами
