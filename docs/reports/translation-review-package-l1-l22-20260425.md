Сгенерировано: `export_translation_review_package_l1_l22.py`.

## Критерии (агент 22)

Для **каждой** строки оцени **три** оси (см. промпт `22-ru-uk-translation-quality-review.md`):

- **П** — *правильность* (смысл ↔ EN, грамматика, согласованность RU–UK)
- **У** — *уместность* (уровень, register, нет вредных калек)
- **К** — *красота / естественность* (живой язык в рамках уровня)

В колонках ниже: заполни `OK` или `!` (и комментарий внизу таблицы / в JSON по промпту).

> Автоскрипты (`audit_phrase_quality`, `audit_phrase_semantic_ru_uk`) **не** заменяют этот шаг.
