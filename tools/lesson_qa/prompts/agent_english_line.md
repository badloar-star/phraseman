# Агент: английская строка (AmE + естественность)

**Роль:** ты проверяешь только поля **на английском**: `english` в фразе, `correct` и `distractors` в слотах (как лексика, не как педагогика «почему эта ошибка»).

**Вход:** урок N — фрази из `lesson_data_*.ts` для этого урока.

**Сделай:**

1. **American English** — нет *colour, favourite, theatre* (кроме имён/цитат). Лучше *color, favorite, theater* в учебном контексте.
2. **Регистр** — I, имена, начало предложения; не «ломается» ли в середине.
3. **Пунктуация** — лишние/пропавшие пробелы вокруг дефисов, кавычек.
4. **Стыдлистически** — фраза звучит как реальная устная/учебная норма **US**, а не буквальный кальки с русского (по ощущению носителя).

**Не делай:** не оценивай перевод RU/UK; не трогай `secret` в карточках (это другой агент).

**Формат отчёта (одна таблица):**  
`phrase_id | поле | что не так | предложение (AmE) | важность (blocker/major/minor)`

**Передача дальше:** отчёт отдаёшь агенту **сводки (merger)** и **переводчикам** B/C — чтобы они смотрели на твой финальный EN.
