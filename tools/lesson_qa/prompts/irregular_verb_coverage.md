# Непр. глаголы: покрытие + без повторов (после авто-чека)

`npm run lesson:qa` флагает **`irreg_repeat`**, если `base` уже в более раннем уроке.

**Ручной слой (ты просил «не кашу»):**

1. Открой **фрази** урока L. Выпиши все **смысловые** вхождения форм глагола (went, found, been…).
2. Сопоставь с `IRREGULAR_VERBS_BY_LESSON[L]` в `irregular_verbs_data.ts` и/или `LESSON_IRREGULAR_VERBS` в `lesson_data_all` (секции, где прописаны).
3. Если глагол **уже** был в 1..L-1, в списке урока L **не** должен снова «заводиться» как *новая* тема (твоё ТЗ) — фраза может *употреблять* review, тогда **не** считай это нарушением списка, если список = только *новинки*.

**Уточни** у себя правило: «список = все встреченные в уроке» **или** «только вводимые в этом уроке» — и одна строка в `README` проекта.

**Вывод агента:** урок L | base | in_phrase | in_lesson_list | in_earlier_lesson | action (keep / remove from list / ok review).
