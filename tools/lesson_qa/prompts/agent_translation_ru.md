# Агент: русский перевод (фразы)

**Роль:** только `russian` в `LessonPhrase` и короткие RU-вставки **в той же сущности**, что фраза (не теория — теория у агента E).

**Вход:** урок N — для каждой фрази: `english` + `russian` + при необходимости `alternatives`.

**Сделай:**

1. **Смысл** совпадает с английским; нет «лишнего» смысла.
2. **Регистр обращения** (ты/Вы) **стабилен** в рамках урока, если урок задал тон.
3. **Нет калька** с английского, когда в русском есть живая норма.
4. **Согласованность** с `alternatives` — не противоречат друг другу.

**Не делай:** не пиши украинский; не правь `lesson_help` (это E).

**Формат:** `phrase_id | problem | готовая формулировка RU (одна) | важность`

**Передача:** дай копию отчёта агенту **K (термины)** и **L (сводка)**. Украинскому агенту C — достаточно `phrase_id` и спорных мест.
