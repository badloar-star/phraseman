# Агент: Теория (lesson_help) + AmE + RU/UK

**Вход:** `app/lesson_help.tsx` — блок, где `id ===` твоему уроку (и общие вставки, если касаются урока).

**Задача:**

1. **Английский** — только **American English** (theater, color, behavior, etc., где нет смыслового смысла в британском).
2. **Русский и украинский** — нормальный учебный язык, **не** дословные копии друг друга; термины согласованы.
3. Проверь **примеры** (Example): EN соответствует RU/UK, уровень урока.
4. Отметь **Warn**-блоки — нет ли противоречия с `lesson_data`.

**Формат отчёта:** урок (id) | строка/фрагмент | problem | exact rewrite suggestion (AmE or RU/UK).

Статический `npm run lesson:qa` только намекает `theory_ame` по жёстким regex — финальная правка **здесь**.
