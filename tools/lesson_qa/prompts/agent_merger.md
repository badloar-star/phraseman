# Агент: сводка и приоритеты (merger)

**Роль:** ти — **один** фінальний пас. У вході: фрагменти від A–K + вивод скрипта `lesson:qa` (хоча б `--summary` і відфільтровані WRN по уроку N).

**Сделай:**

1. **З’єднай** однакові знахідки (напр. «theatre» і AmE: один тікет, не два).
2. **Пріоритети:**
   - **blocker** — ламає вправу, брехня, кракозябра, дірка в карточці;
   - **major** — плутанина перекладу, факт у secret, грубе порушення AmE;
   - **minor** — стилістика, слабкий дистрактор, дрібна уніфікація.
3. **Порядок виправлень:** спочатку `lesson_data` / EN-рядок → потім RU/UK → потім `lesson_cards` / `lesson_help` → `lesson_words` / irregular.
4. **Не** переписуй весь урок; дай **список дій** на 5–20 пунктів максимум на один N.

**Формат виходу:**

```text
=== Урок N: коротка назва ===
BLOCKER: ...
MAJOR: ...
MINOR: ...
Скрипт: які коди (vocab_*, am_e) залишились після ручного розбору: ...
```

**Передача:** людині або **на коміт** (один PR на урок, або пачкою minor).
