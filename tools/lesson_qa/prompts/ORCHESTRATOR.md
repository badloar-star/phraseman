# Оркестратор: как агенты работают вместе

Это **не код** — порядок, в котором ты (или чаты в Cursor) прогоняешь урок **1…32** или один урок **N**, чтобы никто не мешал друг другу и не дублировал работу.

## Роли

| Кто | Файл | Зона |
|-----|------|------|
| **Скрипт** | `npm run lesson:qa` + `README` в `tools/lesson_qa/` | Дыры в данных, кракозябры, карточки vs число фраз |
| **Агент A — английская строка** | `agent_english_line.md` | AmE, естественность EN-фразы |
| **Агент B — русский** | `agent_translation_ru.md` | `russian`, согласованность с EN |
| **Агент C — украинский** | `agent_translation_uk.md` | `ukrainian`, не калька с RU |
| **Агент D — карточки** | `cards_and_secrets.md` | correct/wrong/secret + факты |
| **Агент E — теория** | `theory_ame_ru_uk.md` | `lesson_help` |
| **Агент F — дистракторы** | `distractor_pedagogy.md` | грамматика слота |
| **Агент G — предлоги** | `agent_prepositions.md` | prep vs отдельный раздел |
| **Агент H — словарь урока** | `agent_vocab_dictionary.md` | `lesson_words` vs фразы |

Правила для всех: `00_rules.md`.

## Порядок (один урок N)

1. **Скрипт** — `npm run lesson:qa:summary`, при правках по уроку N — полный лог с фильтром по `L{N}` (вручную).
2. **Параллельно можно** (два чата или один большой блок):
   - A + B + C на **фразы** этого урока (`getLessonData(N)` / кусок `lesson_data_*.ts`).
3. **D** — карточки только для `lessonCards[N]`.
4. **E** — только блок теории для `id=N` в `lesson_help.tsx`.
5. **F** — дистракторы по фразам N.
6. **G** — если в фразах есть `category: preposition`.
7. **H** — словарь: запись `WORDS_BY_LESSON[N]` в `lesson_words.tsx`.
8. **I** — непр. глаголы: строка урока N в `irregular_verbs_data` + фрази.
9. **J** — фрази со сокращениями.
10. **K** — быстрый проход: название урока, повторяющиеся грам. термины.
11. **L** — вшить все MD-отчёты в один: *blocker / major / minor*, без дублей.

## Как «работают вместе» без путаницы

- **A не трогает** UK; **C не трогает** RU (только смысл к EN).
- **D** опирается на **уже согласованные** EN-фразы (после A) и переводы (после B/C), чтобы secret не разошёлся с фразой.
- **E** не переписывает фразы в `lesson_data` — только теорию; если E нашёл противоречие с фразой — **L** помечает «сначала фраза, потом теория».
- **L** в конце гасит дубли: одна и та же проблема не должна идти из B и K дважды.

## Минимальный вариант «мало времени»

Скрипт → **B+C** (переводы) → **D** (карточки) → **L** (сводка). Остальное — по ситуации.
