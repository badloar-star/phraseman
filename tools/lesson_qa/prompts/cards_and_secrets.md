# Агент: карточки + секрет

**Вход:** для урока L открой `lessonCards[L]` в `app/lesson_cards_data.ts` и список фраз `getLessonData(L)`.

**Проверь для каждого индекса 1..N (N = число фраз):**

1. **Соответствие теме** — `correct*` объясняет именно *эту* собранную фразу, а не соседнюю.
2. **wrong\*** — типичная ошибка для *этого* урока, не противоречит grammatically очевидному.
3. **secret\*** — нет сомнительных «фактов»; одна веская идея; RU/UK **один смысл**, не разные мифы.
4. **Тон** — един стиль, без скачка регистра между RU и UK.

**Вывод:** таблица `lesson | idx | field | problem | suggest_fix`.

**Не правь** файл вручную, если в шапке сказано `Auto-generated` — тогда пиши задачу на **генератор/источник** правды.
