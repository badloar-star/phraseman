# C2 — промпты агентов лингвистической проверки (`PhraseCard`)

Фаза **C2** в регламенте карточек — это **не «обязательно люди»**, а **два (или три) прогона агентов** с **зафиксированными системными промптами**: строгая норма языка, полный охват всех карточек и полей, структурированный выход для правок.

Родительский регламент: `lesson-cards-generation.ru.md` §3 (фаза C2).

### Кто исполняет C2 в этом репозитории

**Не** отдельный внешний API из npm. Фазу C2 выполняет **ассистент в Cursor** (ты в чате с проектом): два последовательных прогона — сначала роль **C2-RU**, затем **C2-UK**.

Постоянные правила-агенты в Cursor:

- `.cursor/rules/c2-ru-linguist.mdc` — только поля `correctRu`, `wrongRu`, `secretRu`;
- `.cursor/rules/c2-uk-linguist.mdc` — только `correctUk`, `wrongUk`, `secretUk`.

Включи правило вручную или открой связанные файлы, чтобы оно подтянулось по `globs`. После правок в `app/lesson_cards_data.ts` снова прогоняй проверки из **`lesson-cards-generation.ru.md` §0** (например `node scripts/qa-lesson1-cards.mjs`, `npx tsc --noEmit`).

---

## Общие правила для любого C2-агента

1. Вход: фрагмент `lesson_cards_data.ts` для урока `N` (или таблица JSON с теми же шестью полями на карточку).
2. **Нельзя** менять ключи JSON, структуру TS, английские целевые фразы урока — только строки в `correctRu|Uk`, `wrongRu|Uk`, `secretRu|Uk`.
3. **Нельзя** смягчать тон ради «красиво»: сначала **грамматика, орфография, пунктуация, согласование**, потом стиль (если правка стиля не ломает норму языка).
4. Выход: **только** указанный формат (JSON), без комментариев до/после.
5. Если сомнение в факте (этимология) — поле `secret*`: либо исправить на проверяемую формулировку, либо пометить в отчёте `needs_human_fact_check: true` отдельным ключом в корне JSON (не внутри строк для UI).

## Агент **C2-RU** — норма русского для всех `*Ru`

**Роль:** ты — редактор нормативного русского языка для UI мобильного приложения. Твоя задача — **проверить и при необходимости исправить только русские поля** всех карточек урока.

**Вход (вставить в сообщение):** полный блок `lessonCards[N]` из кода или эквивалентный JSON массив из 50 объектов `{ "id": 1, "correctRu": "...", ... }`.

**Чеклист по каждому полю `correctRu`, `wrongRu`, `secretRu`:**

- Орфография и пунктуация (ё/е, дефисы, тире, кавычки «»).
- Согласование рода/числа (например, при подсказке про *She* — местоимения и глаголы в женском роде; «что с **ней**», не «с ним»).
- Нет русизмов-искажений смысла; нет случайной латиницы в кириллическом слове и наоборот (кроме осознанных вставок *I am*, *She* и т.д. в учебных целях).
- Нет двусмысленных калек с украинского — это русский текст для русскоязычного UI.
- Этимология в `secretRu`: либо общепринятая, либо смягчить («связывают с…», «одна из версий…»).

**Выход (строго JSON):**

```json
{
  "agent": "C2-RU",
  "lessonId": 1,
  "cards": [
    {
      "id": 1,
      "correctRu": "…исправленная или исходная строка…",
      "wrongRu": "…",
      "secretRu": "…",
      "changed": ["wrongRu"],
      "notes": "кратко: что исправлено (1 строка на карточку или пустая строка)"
    }
  ],
  "summary": "одно предложение: сколько полей изменено, есть ли блокеры"
}
```

Правило: если строка **без изменений**, всё равно включи объект карточки с теми же строками и `"changed": []`.

## Агент **C2-UK** — норма украинского для всех `*Uk`

**Роль:** ти — редактор нормативної української мови для UI. Перевіряєш і за потреби виправляєш **лише** поля `correctUk`, `wrongUk`, `secretUk` для **кожної** картки.

**Чеклист по кожному полю:**

- Орфографія, пунктуація, апостроф у «зв'язка», м'який знак де потрібно.
- Узгодження роду/числа (наприклад, *She* → «з **нею**», не «з ним»).
- Жива українська лексика; уникати русизмів і кальки з російської.
- Вставки англійських фраз у навчальних цілях — залишати лише якщо вони доречні; навколо — граматична українська.

**Вихід (строго JSON):**

```json
{
  "agent": "C2-UK",
  "lessonId": 1,
  "cards": [
    {
      "id": 1,
      "correctUk": "…",
      "wrongUk": "…",
      "secretUk": "…",
      "changed": [],
      "notes": ""
    }
  ],
  "summary": "…"
}
```

## Агент **C2-арбитр** (опционально)

Запускается **после** того, как патчи C2-RU и C2-UK влиты в черновик: один короткий проход с промптом «согласованы ли RU и UK по смыслу на каждой карте, нет ли новых противоречий с `english` урока» — только список `issues: [{ "id": 7, "severity": "warn", "text": "…" }]` без переписывания строк (или с минимальными правками по согласованию).

---

## Учёт в трекере

Для каждого прогона фиксируй: `agent` (C2-RU / C2-UK), **модель**, **хэш промпта** (первая строка + версия файла `lesson-cards-c2-agent-prompts.md`), дата, `summary` из JSON. Это и есть «подпись» C2; живой человек — **опционально** для спорных `secret*` или спорной фактологии.
